# **App Name**: QuizWhiz AI

## Core Features:

- Topic Input: Users can enter a specific topic to generate a quiz around.
- AI Quiz Generation: Generative AI tool to create a 10-question quiz based on the topic input.
- Interactive Quiz Interface: Present quiz questions with multiple-choice answers in an engaging, interactive format.
- Instant Feedback: Provide immediate feedback after each question is answered, indicating correctness.
- Results Review: Display a summary of the quiz results, including the score and correct/incorrect answers.

## Style Guidelines:

- Primary color: Vibrant yellow (#FFEA00) for energy and engagement, drawing inspiration from the app's educational, cheerful mood.
- Background color: Soft, desaturated yellow (#FAF7E1) to provide a gentle backdrop that complements the primary color.
- Accent color: Deep purple (#6A00FF) to add a pop of color, and as an analogous color to yellow.
- Body and headline font: 'PT Sans', sans-serif, to achieve a modern yet accessible feel, great for both headings and body text.
- Use clear and intuitive icons for navigation and to represent different quiz elements.
- Incorporate subtle animations and transitions to enhance user experience during quiz-taking and feedback.
- A clean and spacious layout that is easy to follow.
